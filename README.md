A fully functional web-based implementation of the classic **Connect Four** strategy game, built entirely with **ASP.NET Core Blazor**. This project demonstrates how to handle complex game logic, 2D arrays, and interactive UI state using **C#** instead of JavaScript.

* **Classic Gameplay:** Two-player local mode (Red vs. Yellow).
* **Interactive Board:** Click on any column to drop a piece into the lowest available slot.

* **Dynamic UI:** The board updates instantly using Blazor's state management.

## ğŸ“¦ Getting Started

To play the game locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/marius2347/Connect-Four-Game-using-Blazor-in-CSharp.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Connect-Four-Game-using-Blazor-in-CSharp
    ```
3.  **Run the application:**
    ```bash
    dotnet run
    ```
4.  Open your browser and navigate to the localhost URL shown in the terminal (usually `https://localhost:7000`).
